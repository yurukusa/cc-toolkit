# cc-daily-report

AI activity report for your Claude Code Ghost Days.

Shows what your AI accomplished on a specific day — especially useful for **Ghost Days** when you were offline but Claude was still running.

## Usage

```bash
npx cc-daily-report              # yesterday
npx cc-daily-report 2026-02-09   # specific date
npx cc-daily-report 2026-02-09 --tweet   # tweet-ready text only
npx cc-daily-report 2026-02-09 --json    # raw JSON
```

## Output

```
  cc-daily-report  —  2026-02-09
  ────────────────────────────────────────────────

  Status: 👻 Ghost Day

  You  ░░░░░░░░░░░░░░░░░░░░  0.0h
  AI   ███████████████░░░░░  6.0h

  ── Tweet-ready text ──────────────────────
  👻 02/09 のAI活動ログ

  私: 0.0h（不在）
  AI: 6.0h（自律稼働）

  プロジェクト: cc-agent-load, cc-loop

  #ClaudeCode #GhostDay #AI自律稼働

  75 chars (Twitter ~76)  ✓ within limit
```

**Ghost Day** = `main === 0 && sub > 0` — AI ran autonomously, you were offline.

If `~/ops/proof-log/YYYY-MM-DD.md` exists (generated by cc-loop hooks), the report includes per-project session counts and file changes.

## Requirements

- Node.js 18+
- [`cc-agent-load`](https://www.npmjs.com/package/cc-agent-load) installed globally or available in PATH

## Part of cc-toolkit

This tool is part of the [cc-toolkit](https://yurukusa.github.io/cc-toolkit/) collection of Claude Code utilities.

**Zero dependencies.** No data sent anywhere. Runs entirely local.
